# 2.7차 — 세션 탐색기 (Session Explorer)

> Claude Code CLI의 전체 세션을 기존 UI에 통합하여 탐색·검색·복원
> 작성일: 2026-03-17

---

## 1. 배경

현재 웹 UI는 **자체 DB 세션만** 사이드바에 표시한다.
하지만 Claude Code CLI는 `~/.claude/history.jsonl`에 **모든 세션**을 별도 관리하고 있다.
이를 기존 화면에 자연스럽게 통합하면 CLI 세션도 웹에서 탐색·복원할 수 있다.

## 3. 통합 설계

### 방식: 사이드바 탭 전환

사이드바의 세션 목록 영역을 **2개 탭**으로 분리한다.
기존 레이아웃을 유지하면서 CLI 세션을 자연스럽게 노출.

```
┌──────────────────┬─────────────────────────────────┐
│   Sidebar        │   Main Content                  │
│                  │                                 │
│ [Claude Code Web]│  (선택한 세션의 ChatWindow       │
│ [+ 새 세션]      │   또는 CLI 세션 상세보기)        │
│                  │                                 │
│ [내 세션 | CLI]  │                                 │
│  ↕ 탭 전환       │                                 │
```

### "이어서 대화하기" 클릭 흐름

```
1. [이어서 대화하기] 클릭
2. 서버: 새 웹 세션 생성 (name = CLI 세션명, claude_session_id = CLI session ID)
3. 사이드바 탭을 "내 세션"으로 자동 전환
4. 생성된 웹 세션이 활성화됨
5. 첫 메시지 전송 시 --resume {sessionId}로 CLI 컨텍스트 복원
```

## 5. API 설계

| Method | Endpoint | 설명 | Parameters |
|--------|----------|------|-----------|
| GET | `/api/cli-sessions` | CLI 세션 목록 | `?project=&find=&limit=&sort=` |
| GET | `/api/cli-sessions/stats` | 세션 통계 | — |
| POST | `/api/cli-sessions/adopt` | CLI → 웹 세션 연결 | `{ sessionId, sessionName, project }` |

### adopt 응답

```json
{
  "ok": true,
  "data": {
    "session": {
      "id": 25,
      "name": "지금 어떤 프로젝트를...",
      "status": "active",
      "claude_session_id": "a0fb4169-ea21-...",
      "project_path": "/home/forelink/claude"
    }
  }
}
```

## 8. 고려사항

### 기존 UI와의 일관성
- 탭 전환 스타일은 기존 `.mode-toggle`(NewSessionModal)과 동일한 패턴 사용
- CLI 세션 아이템은 기존 `SessionItem`과 유사한 레이아웃 유지
- 상세보기는 기존 `ChatWindow`의 `chat-header-bar` 스타일 차용

### 보안
- CLI 세션은 서버의 `~/.claude/` 에서 읽기 전용
- 모든 사용자에게 동일한 CLI 세션이 보임 (서버 공용 계정)
- adopt 시 work_mode/project_path 검증 필요

### 성능
- history.jsonl을 매 요청마다 파싱하지 않음 → **60초 TTL 캐시**
- JSONL 파일 크기는 `fs.statSync`로 빠르게 조회
- 사이드바에서는 프로젝트당 최근 5개만 표시, "더 보기"로 확장

### 탭 상태 유지
- 탭 선택 상태는 Sidebar 내부 `useState`로 관리
- CLI 세션 클릭 → 상세보기 이동 시 탭 유지
- adopt 후 자동으로 "내 세션" 탭 전환
